SECCIÓN 1 : HTML

En primer lugar, un HTML significa "Hyper Text Markup Language".

Su creador fue Tim Berners-Lee, en el año 1989.

Fue creado para crear, editar y compartir archivos científicos.

Consiste en múltiples etiquetas que funcionan como identificadores, para procesar la información dentro de ellas.

Sin embargo, con el paso del tiempo se fue viendo la necesidad de crear estilos y mejorarlo visualmente.

SECCIÓN 2 : CSS

Es por esto que apareció CSS para mejorar estos archivos preliminares en html.

CSS significa "Cascading Style Sheet.

Es de suma importancia, porque permite lograr editar y agregarle estilos y layouts al HTML plano.

Además, existen muchas ayudas que hoy en día podemos utilizar como "templates" OpenSource.

Uno de los Templates más famosos es Bootstrap (con el que hicimos este Layout).

SECCIÓN 3 : BACKEND

El backend es lo que se encarga de hacer los procesos, métodos y funciones que corren directamente en servidor.

Permite lograr hacer que las páginas web tengan esa capacidad de interactuar y efectuar algoritmos "de fondo".

De esto no sé mucho aún, pero sé que se puede desarrollar en entornos como Java, JavaScript, Python, PHP, C, etc.
